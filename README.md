# 🧱 Item Hierarchy App  

A **web-based application** built using **ASP.NET Core MVC** that allows users to manage items in a **Parent–Child hierarchy structure**.  

---

## 📌 Overview  

The **Item Hierarchy App** is designed to organize and manage structured data where items can have relationships (parent and children).  

It allows users to:
- Create items  
- Link them hierarchically  
- Search and manage data efficiently  

This project demonstrates **CRUD operations + hierarchical data handling + clean UI design**.

---

## 🛠 Tech Stack  

- 💻 ASP.NET Core MVC  
- ⚙️ C#  
- 🗄 Entity Framework Core  
- 📦 SQLite Database  
- 🎨 Bootstrap (UI Styling)  
- 🧩 Razor Views  

---

## 🚀 Features  

- ➕ Add new items  
- ✏️ Edit items  
- ❌ Delete items (with validation)  
- 🔍 Search items by name  
- 🌳 Add child items to a parent  
- 📊 Display hierarchical relationships  
- 🧩 Tree view representation  
- ✅ Input validation (Name & Weight)  

---

## ⚙️ How It Works  

1. User creates an item (Name + Weight)  
2. Items can be linked as **Parent → Children**  
3. Data is stored using **SQLite database**  
4. Search filters items dynamically  
5. Tree view shows hierarchical structure
